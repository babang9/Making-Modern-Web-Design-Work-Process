### [Gulp](http://gulpjs.com) 글로벌 설치
```sh
$ npm i -g gulp
```
-

### Gulp 로컬 설치
```sh
$ npm i -D gulp
```

-

### Gulp 업무 실행
```sh
$ gulp {업무이름}
```
